#markdown  基础知识

1.段落和换行： 空格或者<br/>

2.标题：(1)文字后加 = 最高阶标题和 - 第二阶标题

this is h1
==========

this is h2
----------

(2)利用1到6个#；

3.区块引用：> 或者嵌套引用>>

4.列表：无序列表多个 *  +  -     有序列表：1. 2.   或 ol li的html标签

5.代码:4个空格或者1个制表符；或则pre标签，code标签

6.分割线：三个以上的星号、减号、底线来建立一个分隔线

7.链接：链接文字都是用 [方括号] 来标记，后面用冒号表示

8.强调：用星号和底线（_）表示

9.图片：一个惊叹号 !,接着一个方括号，里面放上图片的替代文字,接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上 选择性的 'title' 文字
